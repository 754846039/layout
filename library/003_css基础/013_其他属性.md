#### 鼠标样式cursor
属性值|描述
---|---
hand |手型
pointer| 手型（这里推荐使用这种，因为这可以在多种浏览器下使用）
crosshair | 十字型
text| 移动到文本上的那种效果
wait |等待的那种效果
default| 默认效果
help |问号
e-resize| 向右的箭头
ne-resize| 向右上的箭头
n-resize| 向上的箭头
nw-resize| 向左上的箭头
w-resize |向左的箭头
sw-resize| 左下的箭头
s-resize |向下的箭头
se-resize| 向右下的箭头
auto| 由系统自动给出效果

#### resize属性
主要处理 textarea 的缩放
```css
resize:none;
resize:both;
resize:horizontal;
resize:vertical;
```
