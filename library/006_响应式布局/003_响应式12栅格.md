## 响应式十二栅格系统
- 主要用来勾勒草图，将页面快速分块
```css
.container{
	width: auto;
	height: auto;
	/*因为在栅格化布局中，如果每一列都有内边距padding，想把其中一块再分成两块，可以直接在里面直接写一个row,不用写container，不用担心边距问题，如果不设置padiding,会出现双倍边距问题*/
    /*如果没有嵌套关系，且每列之间没有间距，不写padding也可以*/
	padding:0 15px;
}
.row{
	width: auto;
	height: auto;
	/* 多层嵌套设置padding值时，如果container不设置padding,row不设置margin值，会出现双倍边距 */
	margin: 0 -15px;
}
/* 委元素选择器 */
.row:after{
	content: "";
	display: block;
	clear: both;
}
/* 属性选择器 */
[class*="col-"]{
	/* 如果需要两个1/3的列左右排列，则可以写三个col-xs-4，min-height会把元素撑开 */
	min-height: 1px;
	float: left;
}
/* 定义最小屏下的类 */
/* 如果页面中没有响应式，不管是大屏小屏都是一样的布局，那么只需要用col-xs即可，在任何情况下都会起作用 */
.col-xs-1{
    width: 8.3333%;
}
...
/* sm 小屏  768-922 */
@media screen and (min-width:768px) {
    .col-sm-1{
        width: 8.3333%;
    }
    ...
}
/* md 中屏  922-1200 */
@media screen and (min-width:992px) {
    .col-md-1{
        width: 8.3333%;
    }
    ...
}
/* lg 大屏  >1200 */
@media screen and (min-width:1200px) {
    .col-lg-1{
        width: 8.3333%;
    }
    ...
}
```
