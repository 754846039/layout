# 列表元素
## 无序列表
```html
<ul type="disc">
<!--type="disc"     实心小圆圈 (默认样式)-->
<!--type="circle    空实心小圆圈 -->
<!--type="square"   小方块-->
    <li>列表1</li>
    <li>列表2</li>
</ul>
```

## 有序列表
```html
<ol type="1">
<!--type="1/a/A/i/I"-->
    <li>列表1</li>
    <li>列表2</li>
</ol>
```

## 自定义列表
```html
<dl>
    <dt>列表条目</dt>
    <dd>列表定义、解释</dd>
    <dd>列表定义、解释</dd>
</dl>
```
